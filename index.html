Commands List:

Administrator Commands

These are commands which can only be used by users with adminstrator permissions:

v!createElection Election Name :Creates an election on your server. You must give an election name

v!endElection Election Name:Ends and election. When invoked the election results will be displayed in the channel the command was run on

v!audit Election Name:Sends all the votes in the election to the channel the command was run on.This command is only to be used to check for suspicious votes

User Commands

v!join Election Name: Allows you to join an election as a candidate

v!dropOut Election Name: Allows you to drop out of an election, if you are a candidate

v!vote Election Name: Allows a candidate to vote. When invoked you will recieve a direct message from the bot witch a list of candidates. You respond with the name of the candidate you wish to vote for.

v!status Election Name: Provides an updated list on the vote count of an election.

v!list : Returns a list of the names of all the elections on the server
